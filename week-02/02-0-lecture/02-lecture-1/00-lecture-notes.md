## Floating point numbers
Internal structure of floating point values

*Single precision*
| sign | exponent | fraction |
| :--: | :------: | :------: |
| 1 bit | 8 bits | 23 bits |
| 31 | 30 - 23 | 22 - 0 | 

*Double precision*
| sign | exponent | fraction |
| :--: | :------: | :------: |
| 1 bit | 11 bits | 52 bits |
| 63 | 62 - 52 | 51 - 0 |
